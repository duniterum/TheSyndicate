# Product OS Studio

This folder is an isolated Replit Product OS prototype workspace.

It is not production.

It is not the live website.

It is not smart-contract authority.

It is not auth authority.

It may contain simulated/future UI.

It may contain localStorage role simulation.

It may contain mock data.

It may contain future modules.

It must never be treated as production truth without Codex production integration.

## Ownership Rules

- Replit 2 may own visual/product development inside this folder.
- Codex may add adapters, interfaces, bridge contracts, tests, and production-readiness notes.
- Codex must not redesign Replit's UI without founder approval.
- Production app must not import from this folder unless a future approved integration sprint explicitly does so.

## Production Boundary

- No real wallet writes.
- No real founder/admin auth.
- No source/referral activation.
- No real claim UI.
- No fake explorer links as live.
- No fake addresses as live.
- No production deployment from this folder.
- All future/simulated surfaces must be labeled.

## Current Use

Use this folder as the safe landing zone for the standalone Replit 2 Product OS Studio prototype. Keep production truth in the production app until a later approved Codex bridge sprint defines, tests, and releases a specific slice.
