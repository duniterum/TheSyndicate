### Why The Root Is Not Final-Frozen Here

The V3 root must be generated from numbered historical-member leaves, not address-only leaves. The exact proof artifact must be generated by committed, repeatable tooling or a reviewer-accepted equivalent before any broadcast approval.

During this preparation pass, the live read-only scan confirmed the member roster did not change. However, the currently committed `contracts/tools` scripts are V2/V2b-era scripts and do not provide a committed V3 numbered-root generator for V1 + V2a + V2b. An attempted isolated local install of the OpenZeppelin Merkle helper failed/timed out, so this package does **not** claim a final root freeze.

Broadcast remains blocked until the final root/proof artifact is generated, hashed, and verified.
