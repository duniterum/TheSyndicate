* System Wallet Action

D) Artifact vs Memory vs Seat

* Archive1155
* First Signal
* Patron Seal

* chapter artifacts
* milestone artifacts

NFTs are artifacts, not automatically seats.

* Share Object
* OG Card
* Public Wallet Page
* Milestone Page
* Referral attribution

Referral starts as attribution only, not reward.

* Vault
* Liquidity
* Operations
* LP
* protocol-owned assets
* member-held assets

Treasury visibility is transparency, not financial return.
