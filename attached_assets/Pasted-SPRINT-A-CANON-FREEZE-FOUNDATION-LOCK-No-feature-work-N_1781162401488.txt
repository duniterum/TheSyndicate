FOUNDATION LOCK

No feature work.

No redesign.

No refactor.

No contract work.

No new modules.

No new economics.

No public UI changes.

No implementation beyond canon/documentation updates if required.

Goal:

Freeze the doctrine of The Syndicate so future work cannot drift.

This sprint exists because the protocol has reached sufficient complexity that the primary risk is no longer missing features.

DRIFT.

Future systems must plug into the same architecture.

## ADJACENCY LAW

Propose or canonize:

Each layer may only consume the layer directly above it.

Example:

EVENTS consume TRUTH.

SIGNALS consume EVENTS.

MEMORY consumes SIGNALS + EVENTS.

STORY consumes MEMORY.

No layer should skip a layer.

If a feature requires skipping layers:

it is likely misplaced.

Evaluate whether this should become constitutional.
